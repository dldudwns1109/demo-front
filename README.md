<h1 align="center">🎉 DE:MO</h1>
<p align="center">파이널 프로젝트 DE:MO 리파지토리입니다.</p>

DE:MO(대학생 모임)는 대학생들이 관심사 기반으로 소모임을 자유롭게 개설하고 참여할 수 있는 대학생 전용 커뮤니티 플랫폼입니다.
